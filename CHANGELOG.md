# Changelog

## v1.2.0 — 2026-07-02

### Added
- Manual Hide Week / Show Week controls for individual calendar rows
- Browser memory for hidden weeks, separated by page and month
- Floating changelog button and in-app version history
- Frontend save lock and backend duplicate-request protection

### Preserved and Improved
- Time-based automatic sorting
- Quick status-update icons
- Floating refresh button
- Live Manila date and time in the upper-right
- Dynamic managed-page tabs
- Lighter Inter typography and fast document-fragment rendering

## v1.1.0 — 2026-06-21

### Added
- Time field for posts
- Automatic sorting by posting time
- Quick status-update icon on calendar cards
- Floating refresh button
- Live Manila time and date
- `config.js` for centralized configuration
- Footer version and refresh timestamp
- Toast notifications
- Request timeout handling
- Apps Script support for the `Time` column
- README and setup documentation

### Improved
- Lighter Inter typography with reduced blockiness
- Faster DOM rendering using document fragments
- More resilient Apps Script header mapping
- Cleaner mobile behavior
- Better loading and disabled-button states

## v1.0.0

- Initial DesignLab Tracker calendar
- Dynamic page tabs
- Add, edit, delete posts
- Google Sheets and Apps Script integration
